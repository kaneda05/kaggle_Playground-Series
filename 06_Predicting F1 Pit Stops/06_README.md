# 🏎️ F1 Pit Stop Prediction (Playground Series - S6E5) コンペティション概要

**公式リンク:** [Kaggle Competition Page](https://www.kaggle.com/competitions/playground-series-s6e5)

## 🏆 目的 (Goal)
F1（フォーミュラ1）のレースデータを用いて、ドライバーが**「次のラップでピットインするかどうか（`PitNextLap`）」**を予測する二値分類タスクです。

## 📊 データセットについて (Dataset)
本コンペティションのデータは、「F1戦略データセット（F1 strategy dataset）」をベースに生成された**合成データ**です。

*   **データ特性:** 特徴量の分布はオリジナルに近いですが、完全に同一ではありません。
*   **意図的な変更点:** オリジナルデータに存在した `Normalized_TyreLife` という特徴量は、予測が簡単になりすぎる（答えが自明になってしまう）ため、意図的に除外されています。
*   **外部データの許可:** オリジナルのデータセットを外部データとして使用することが**許可・推奨**されています。オリジナルと合成データの違いを分析したり、学習に組み込んで精度向上を図ることがコンペの鍵となります。

## 📈 評価指標と提出形式 (Evaluation & Submission)
*   **予測対象:** テストデータセットの各 `id` に対して、ターゲット変数 `PitNextLap` が1（ピットインする）となる**確率**を予測します。
*   **提出フォーマット:** ヘッダー行を含み、`id` と `PitNextLap` の2つの列を持つCSVファイル形式で提出します。

**提出ファイルの例（`submission.csv`）:**
```csv
id,PitNextLap

439141,0.82
439142,0.03
